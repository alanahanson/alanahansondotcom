Projects

grouVie

March 2016

Github repo

grouVie is a mobile app designed for friends to create group challenges and hold each other accountable in meeting goals. The app, built in React Native with a Rails backend, incorporates an instant messaging feature as well as photo sharing and camera access. grouVie was conceived, researched, and built in 8 days as a final group project at Dev Bootcamp.

Marvellist

February 2016

Marvellist is a web app that uses the Marvel Comics API to create a more usable reading experience for comic fans. It allows users to build a Netflix-style queue to keep track of their digital comic subscriptions and to mark where one is in the series reading order . The original app was built in 2 days using Sinatra, Ruby, and JavaScript.
